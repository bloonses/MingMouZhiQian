# 🌐 校园网访问要求说明

## 📋 网络环境要求

### ✅ 需要连接校园网的情况

**如果您希望学生用户可以通过以下方式访问系统：**
- 在学校教室里访问
- 在实验室里访问  
- 在校园内任何地方访问
- 使用手机扫码使用考勤功能

**那么您必须：**
- 🔌 将运行系统的电脑连接到校园网
- 🖥️ 确保防火墙允许5000端口访问
- 📱 确保同一校园网内的其他设备可以访问

### ❌ 不需要校园网的情况

**如果您只需要：**
- 在本地运行系统进行测试
- 只在您自己的电脑上使用
- 不需要其他用户访问

**那么：**
- ✅ 可以直接在本地运行
- ✅ 用户可以通过 `localhost:5000` 访问
- ✅ 不需要连接校园网

## 🏫 不同使用场景

### 场景1：教室考勤（需要校园网）
```
您的电脑 (连接校园网) ── 校园网 ── 学生手机/电脑
    ↓
系统运行在 http://[您的IP]:5000
学生通过扫码或输入地址访问
```

### 场景2：本地测试（不需要校园网）
```
您的电脑 (本地网络) ── 本地访问
    ↓  
系统运行在 http://localhost:5000
只有您可以访问，用于测试
```

## 🚀 如何判断是否需要校园网

### 需要校园网的情况：
- ✅ 您希望学生能在教室里使用系统
- ✅ 您希望学生能扫码签到
- ✅ 您希望多个用户同时访问
- ✅ 您需要在校园内任何地方使用

### 不需要校园网的情况：
- ✅ 您只是想测试系统功能
- ✅ 您只需要在本地使用
- ✅ 您不打算让其他用户访问
- ✅ 您还在开发调试阶段

## 🔧 技术说明

### 校园网内访问原理：
```
您的电脑IP: 192.168.1.100 (校园网内)
系统端口: 5000
学生访问: http://192.168.1.100:5000
```

### 本地访问原理：
```
您的电脑IP: 127.0.0.1 (本地)
系统端口: 5000
您访问: http://localhost:5000 或 http://127.0.0.1:5000
```

## 📱 实际使用建议

### 如果您需要在校园网内使用：

1. **连接校园网**：
   - 使用网线连接校园网
   - 或连接校园Wi-Fi
   - 确保网络正常

2. **运行系统**：
   ```bash
   python app_campus.py
   ```

3. **获取访问地址**：
   - 系统会显示您的IP地址
   - 学生可以通过该地址访问

4. **学生访问方式**：
   - 直接输入IP地址
   - 或扫描二维码

### 如果您只是本地测试：

1. **无需连接校园网**
2. **直接运行**：
   ```bash
   python app_campus.py
   ```
3. **访问地址**：
   - http://localhost:5000
   - http://127.0.0.1:5000

## 📞 网络问题排查

如果校园网内无法访问：

1. **检查网络连接**：
   ```bash
   ping 8.8.8.8
   ```

2. **检查端口是否占用**：
   ```bash
   netstat -an | findstr 5000
   ```

3. **检查防火墙设置**：
   - 允许5000端口入站连接
   - 暂时关闭防火墙测试

4. **获取正确的IP地址**：
   ```bash
   ipconfig  # Windows
   ifconfig  # Linux/Mac
   ```
