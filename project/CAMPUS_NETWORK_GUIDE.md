    host='0.0.0.0',

### 自定义静态文件路径
```python
# 配置静态文件路径
app.config['STATIC_FOLDER'] = 'static'
app.config['TEMPLATE_FOLDER'] = 'templates'
```

### 设置访问限制
```python
# 添加访问控制
@app.before_request
def limit_access():
    # 只允许校园网访问
    if not request.remote_addr.startswith('192.168.'):
        return jsonify({'error': '访问被拒绝'}), 403
```

## 📞 故障排除

### 常见问题

#### 1. 无法访问系统
**可能原因**：
- 防火墙阻止端口访问
- 网络配置问题
- 系统未正确启动

**解决方案**：
```bash
# 检查防火墙设置
# Windows: 控制面板 → 系统和安全 → Windows Defender 防火墙
# 允许5000端口入站连接

# 检查系统是否在运行
# 查看端口是否被占用
netstat -an | findstr 5000
```

#### 2. 用户无法通过IP访问
**可能原因**：
- 路由器设置限制
- NAT配置问题
- 网络隔离

**解决方案**：
```bash
# 尝试使用localhost访问
http://localhost:5000

# 检查网络配置
ipconfig
# 或 ifconfig
```

#### 3. 二维码无法使用
**可能原因**：
- 网络协议问题
- 移动设备兼容性

**解决方案**：
```bash
# 直接使用URL访问
# 生成新的二维码
# 确保使用http协议
```

#### 4. 云存储功能异常
**可能原因**：
- 网络连接问题
- 配置文件错误

**解决方案**：
```bash
# 检查云存储配置
python test_quick_b2.py

# 检查网络连接
ping backblazeb2.com
```
