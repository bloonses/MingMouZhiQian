#### 后端模式（InsightFace）：⭐新

**face_recognition_backend.py 核心代码**：

```python
class LivenessTracker:
    """鼻尖移动活体检测"""
    def __init__(self):
        self.nose_history = []
        self.max_history = 25
        self.min_frames = 6
        self.movement_threshold = 10
    
    def update(self, nose_tip):
        self.nose_history.append({'x': float(nose_tip[0]), 'y': float(nose_tip[1])})
        if len(self.nose_history) > self.max_history:
            self.nose_history = self.nose_history[-self.max_history:]
    
    def is_live(self):
        if len(self.nose_history) < self.min_frames:
            return False
        recent = self.nose_history[-self.min_frames:]
        y_values = [p['y'] for p in recent]
        x_values = [p['x'] for p in recent]
        y_range = max(y_values) - min(y_values)
        x_range = max(x_values) - min(x_values)
        return y_range > self.movement_threshold or x_range > self.movement_threshold

def cosine_similarity(a, b):
    """余弦相似度计算"""
    a_norm = np.linalg.norm(a)
    b_norm = np.linalg.norm(b)
    return np.dot(a, b) / (a_norm * b_norm)
```

```python
class LivenessTracker:
    def __init__(self):
        self.nose_history = []
        self.max_history = 25
        self.min_frames = 6
        self.movement_threshold = 10
    
    def update(self, nose_tip):
        self.nose_history.append({
            'x': float(nose_tip[0]),
            'y': float(nose_tip[1])
        })
        if len(self.nose_history) > self.max_history:
            self.nose_history = self.nose_history[-self.max_history:]
    
    def is_live(self):
        if len(self.nose_history) < self.min_frames:
            return False
        recent = self.nose_history[-self.min_frames:]
        y_values = [p['y'] for p in recent]
        x_values = [p['x'] for p in recent]
        y_range = max(y_values) - min(y_values)
        x_range = max(x_values) - min(x_values)
        return y_range > self.movement_threshold or x_range > self.movement_threshold
```

## ⚠️ 常见问题与解决方案（V2.1）

### 问题1：face-api.js模型加载失败
**原因**：模型文件路径错误或文件缺失
**解决**：
- 确保模型文件在 `/static/models/` 目录下
- 检查浏览器控制台的网络请求
- 确认所有模型文件都已下载

### 问题2：InsightFace模型加载失败 ⭐新
**原因**：模型目录不对、numpy版本太高、onnxruntime未安装
**解决**：
- 确保模型文件在 `.insightface/models/buffalo_l/` 目录下
- 确保模型文件直接在buffalo_l下，不要嵌套
- 确保numpy<2.0：`pip install "numpy<2.0"`
- 确保onnxruntime已安装：`pip install onnxruntime`
- 参考 [INSIGHTFACE_DEPLOYMENT.md](INSIGHTFACE_DEPLOYMENT.md)

### 问题3：摄像头无法访问
**原因**：浏览器权限问题、HTTP协议限制、facingMode错误
**解决**：
- 使用localhost或127.0.0.1访问
- 在浏览器设置中允许摄像头权限
- 检查是否有其他程序占用摄像头
- 确保facingMode是'user'不是'environment'

### 问题4：人脸识别不准确（前端模式）
**原因**：光线不足、人脸角度问题、阈值不合适
**解决**：
- 确保采集和识别时光线充足
- 人脸正对摄像头
- 可以适当降低匹配阈值（例如从0.55降到0.5）
- 参考 [关键参数文档.md](关键参数文档.md)

### 问题5：人脸识别不准确（后端模式）⭐新
**原因**：余弦相似度阈值不合适、活体检测未通过、未采集512维特征
**解决**：
- 检查终端日志中的cos_sim值
- 确认采集了512维特征（采集时提示"双特征"）
- 确认活体检测通过了（终端显示liveness=True）
- 如果cos_sim低于0.45，考虑降低阈值
- 参考 [关键参数文档.md](关键参数文档.md)

### 问题6：点头检测没反应
**原因**：阈值不合适、距离太远、动作不明显
**解决**：
- 点击"🔧 调试"按钮查看详细数据
- 调整相关参数（参考参数文档）
- 确保距离合适、动作自然

### 问题7：切换班级/模式后画面不动 ⭐新
**原因**：ctx为null、canvas未重置
**解决**：
- 已修复！切换时重新初始化canvas和ctx
- 确保视频流正常播放

### 问题8：数据库操作报错
**原因**：表结构不匹配或数据类型错误
**解决**：
- 删除数据库文件重新创建
- 检查模型定义与数据库是否一致
- 使用 `db.create_all()` 重新创建表
- 数据库会自动添加face_descriptor_512列
